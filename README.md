Learning Center Database (SQL Project)

Overview
This project implements a relational database system for managing a learning center, including students, courses, instructors, enrollments, and payments.

Features
- Database design with ER diagram
- Multiple related tables (students, courses, schedules, payments...)
- Data integrity using Primary Keys and Foreign Keys
- Real-world relationships (many-to-many, one-to-many)

 SQL Concepts Used
- CRUD operations (INSERT, SELECT, UPDATE, DELETE)
- Joins (INNER, LEFT, RIGHT)
- Aggregate functions (COUNT, AVG, SUM)
- Subqueries and nested queries
- Window functions (RANK)
- Triggers for data tracking

 Tools
- SQL Server Management Studio

 Files
- SQL scripts for table creation and data insertion
- ER diagram
- Project report
- Query analysis and outputs
